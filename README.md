# BPO-repeat-contact-analysis
Project Summary

This project analyzes repeat customer contact behavior in customer support operations using a DMAIC framework. The goal is to identify root causes of repeat interactions and propose process improvements to reduce operational cost and improve resolution quality.

Key Insights
~21% of tickets are repeat contacts
Faster resolution is associated with higher repeat rates
High-volume categories drive the largest cost impact
CSAT does not reflect operational inefficiency

Tools Used

Python (Pandas),
Tableau,
Notion
